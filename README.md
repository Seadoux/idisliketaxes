# IDislikeTaxes

Tax deduction finder for YouTubers and content creators.

## Features

- Upload bank statements (CSV or PDF)
- Automatic expense categorization
- Schedule C line-by-line breakdown
- YouTube-specific merchant database

## Tech Stack

- React + Vite
- Tailwind CSS
- Stripe payments
- PDF.js for statement parsing

## Setup

```bash
npm install
npm run dev
```

## Deploy

Deploy to Vercel:
```bash
vercel --prod
```
